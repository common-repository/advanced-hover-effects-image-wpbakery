== Installation ==

Installing this plugin as regular wordpress plugin. 



    The easy way:

        Go to the Plugins Menu in WordPress
        Search for “ CSS Image Hover Effects for WPBakery Page Builder  ”
        Click “Install”

    The not so easy way:

        Upload the ml-slider folder to the /wp-content/plugins/ directory
        Activate the plugin through the ‘Plugins’ menu in WordPress
        Manage your slideshows using the ‘ CSS Image Hover Effects for WPBakery Page Builder’ menu option
